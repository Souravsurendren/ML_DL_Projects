# Analyzing Rainfall Trends in India

This project analyzes historical rainfall data in India to uncover trends, seasonal patterns, and regional variations. Using data science and visualization tools, it helps understand how rainfall has evolved over time, aiding climate studies and agricultural planning.

## About the Project

India’s rainfall varies dramatically by region and season, largely influenced by the monsoon. This project provides an in-depth analysis of rainfall trends across Indian states and years using a data-driven approach.

Key objectives:
- Analyze monthly, seasonal, and annual rainfall.
- Compare rainfall across different states and regions.
- Visualize anomalies and detect long-term patterns.

---

## Dataset

The dataset contains monthly and yearly rainfall data for Indian states.

Typical columns:
- `YEAR`
- `STATE/REGION`
- `JAN`, `FEB`, ..., `DEC`
- `ANNUAL`
